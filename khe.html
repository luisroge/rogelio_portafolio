Khe Compras?

La aplicación me tomo en desarrollarla alrededor de unos 3-4 meses. Se desarrollo en Android Studio, utilizando Java, Firebase, NOSQL y SQLite.

Base de Datos

Se utilizo para Firebase de Google para guardar los datos. Se opto por NOSQL debido que la base de datos no era muy complicada y las consultas iban ser mas cortas y fáciles.

Pantalla de Inicio

Pagina de ingreso a la aplicación. Para ingresar a la aplicación se debe registrar primero.

Pantalla de Registro

En esta pantalla los usuarios se pueden registrar para ingresar a la aplicación.

Hamburger Menu

Al ingresar el usuario podrá mirar la ofertas del día y también podrá acceder al menú del app en donde podrá ver diferentes opciones.

Catalogo de Productos

En el Catálogo se podrá ver los diferentes tipos de productos que se esta ofreciendo.

Busqueda de Productos

Pantalla de Búsqueda donde muestra los productos buscados por el usuario.

Carrito de Producto

Pantalla donde muestra el carrito y todos los productos que el usuario eligió.
