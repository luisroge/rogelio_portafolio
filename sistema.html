SISTEMA DE REGISTRO, CONTROL Y SEGUIMIENTO ESTADISTÍCO DE ENTREGA DE TRABAJOS DE GRADUACIÓN

Lo primordial antes de empezar la elaboración de la aplicación es comprender el problema que el cliente tiene y como una aplicación web puede ayudarlo. Se debe ofrecer una aplicación web que ofrezcan los servicios que se ajusten a las necesidades del cliente. Se tiene un compromiso con el cliente en encontrar los requerimientos para la aplicación. Se aplico la metodología ágil que es recomendada para los trabajos de entorno web. Se separaron los diferentes requerimientos entre funcionales y no funcionales. Realice los diferentes casos de uso para cade interacción del usuario con la aplicación. Analice el costo de proyecto y la herramienta que usarían. Realice las diferentes pruebas y verifique su aceptación con el cliente.
